# alt-text-via-llm

Experimental project to use LLMs on Oscar to create alt-text for images and potentially video description for videos.

## Overview

This project aims to leverage Large Language Models (LLMs) running on Oscar (Brown University's high-performance computing cluster) to automatically generate:
- Alt-text for images
- Video descriptions for videos

## Status

This is an experimental project currently in early development.
